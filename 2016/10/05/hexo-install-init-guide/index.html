运行Hexo静态站点

Hexo安装完成后，在初始化站点下使用如下命令集合运行起来：

  1. hexo g：编译生成静态资源文件
  2. hexo s：运行hexo
  3. hexo new post postName：该命令会在your-hexo-site\source_posts文件夹下生成一个叫postName的博客文件。
