运行一个docker容器

注意：
-d：后台运行docker容器。
-p：端口映射。宿主机port:容器port，如此在访问宿主机的ip+port时，就能访问到docker容器中的 应用端口。
--name：容器名称, 容器id难以记忆，因此可以通过容器名称对容器操作。

查看tomcat启动情况
